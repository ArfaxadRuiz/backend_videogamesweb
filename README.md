Videojuegos API (Backend)

API REST desarrollada con Django + Django REST Framework para la gestión de videojuegos por usuario.
Este backend permite a los usuarios autenticarse mediante JWT y administrar su colección de juegos (crear, ver, actualizar y eliminar).

Tecnologías utilizadas

Python 3
Django
Django REST Framework
JWT Authentication (SimpleJWT)
PostgreSQL
Docker & Docker Compose

Funcionalidades

Autenticación con JWT (login y refresh token)
Gestión de usuarios
CRUD de videojuegos
Relación de juegos por usuario
Protección de endpoints con autenticación
